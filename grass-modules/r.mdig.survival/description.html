DESCRIPTION

r.mdig.survival - Applies change mortality to populations (raster cells) within an input map based on either a single parameter probability of survival or based on the values within a survival map. The module replaces cells that don't survive with null values.
(GRASS Shell Script)

NOTES

Chance of survival should be an integer between 0 and 100.
