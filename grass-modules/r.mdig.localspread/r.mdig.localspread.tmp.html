NAME

r.mdig.localspread - Local spread to neighbouring cells based on a specified spread rate,or based on raster map with spread rates for each cell.

KEYWORDS

SYNOPSIS

r.mdig.localspread
r.mdig.localspread help
r.mdig.localspread [-boz] input=string output=string [spread=float] [spreadmap=string] [proportion=float] [--verbose] [--quiet]

Flags:

-b
Boolean spread, cells are present/absent
-o
Overwrite output file if it exists
-z
Explicitly check and ignore cell values that are zero.
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

input=string
Name of an input layer
output=string
Name of an output layer
spread=float
Spread in metres
spreadmap=string
Raster map containing spread rates in metres
proportion=float
Proportion of individuals to spread from destination cell
Default: 1.0
